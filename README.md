# Fashion AX Claude Toolkit 🌟

> Korea's first open-source Fashion AI Transformation (AX) resource built on Claude.

## Overview

This repository provides prompt libraries, workflow templates, and domain-specific 
AI toolkits for the Korean fashion industry — developed in partnership with the 
Seoul AI Hub CEO Program and Gyeonggi AI Membership Program.

## Planned Contents

- 👗 **Trend Analysis Prompts** — Claude-powered fashion trend forecasting templates
- 🛍️ **Merchandising Workflows** — AI-assisted product planning and assortment tools
- 📊 **Consumer Behavior Modeling** — Prompt frameworks for customer insight analysis
- 🎓 **Education Curricula** — Open learning materials for students and educators

## Background

Developed by Korea's first and only Fashion AI & AX education company.  
Currently used in corporate training programs with major Korean fashion enterprises.  
Part of an initiative to make Claude accessible to fashion students and professionals nationwide.
